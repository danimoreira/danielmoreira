Experiência Profissional

No mercado financeiro, atuou no desenvolvimento de soluções desktop (backoffice de várias Instituições Financeiras) e também soluções Web, principalmente para concessão de crédito para Pessoa Física. Um sistema complexo, que tinha como premissa executar 24x7. Neste projeto foi utilizada a tecnologia PowerBuilder, VB.Net e os gerenciadores de banco de dados Oracle, SQL Server e Sybase.

Outra experiência no mercado financeiro, também foi a passagem pelo Banco BMG, onde atuou desenvolvendo soluções para automatizar o processo de Crédito Consignado, que é o carro chefe da instituição.
